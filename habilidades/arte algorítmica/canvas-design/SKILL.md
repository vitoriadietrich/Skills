<h1>Canvas Design Skill</h1>

<p><b>Create visual art in .png or .pdf using design philosophy.</b></p>

</div>

<hr>

<h2>Skill Metadata</h2>

<pre>
---
name: canvas-design

You should use this skill when the user asks to create a poster, piece of art, design, or other static piece. </pre>

<hr>

<h2>Overview</h2>

<p>
This skill creates aesthetic movements and expresses them visually.
</p>

<p>Output formats:</p>

<ul>
<li>.md – Design philosophy</li>
<li>.pdf – Canvas artwork</li>
<li>.png – Canvas artwork</li>
</ul>

<p>The process happens in two stages:</p>

<ol>
<li><b>Design Philosophy Creation</b> (.md)</li>
<li><b>Canvas Expression</b> (.pdf or .png)</li>
</ol>

<hr>

<h2>Design Philosophy Creation</h2>

<p>
Create a visual philosophy interpreted through:
</p>

<ul>
<li>Form, space, color, composition</li>
<li>Images, graphics, shapes, patterns</li>
<li>Minimal text used only as visual accent</li>
</ul>

<h3>Critical Understanding</h3>

<ul>
<li>User input is only a conceptual seed</li>
<li>The output is an aesthetic movement</li>
<li>The philosophy is later expressed visually</li>
</ul>

<p>
The visual artifact should be <b>90% design and 10% essential text.</b>
</p>

<hr>

<h2>Generating the Visual Philosophy</h2>

<h3>Name the Movement</h3>

<p>Examples:</p>

<ul>
<li>Brutalist Joy</li>
<li>Chromatic Silence</li>
<li>Metabolist Dreams</li>
</ul>

<h3>Articulate the Philosophy</h3>

<p>
Write <b>4–6 paragraphs</b> describing the aesthetic in terms of:
</p>

<ul>
<li>Space and form</li>
<li>Color and material</li>
<li>Scale and rhythm</li>
<li>Composition and balance</li>
<li>Visual hierarchy</li>
</ul>

<hr>

<h2>Critical Guidelines</h2>

<ul>
<li>Avoid redundant explanations</li>
<li>Emphasize craftsmanship repeatedly</li>
<li>Leave creative room for visual interpretation</li>
<li>Focus on visual communication rather than text</li>
</ul>

<p>
The final artwork should feel <b>meticulously crafted and executed at master level.</b>
</p>

<hr>

<h2>Essential Principles</h2>

<ul>
<li><b>Visual Philosophy</b> – create an aesthetic worldview</li>
<li><b>Minimal Text</b> – text used only as visual element</li>
<li><b>Spatial Expression</b> – ideas communicated through form and composition</li>
<li><b>Artistic Freedom</b> – allow interpretive visual execution</li>
<li><b>Pure Design</b> – create art objects, not decorated documents</li>
<li><b>Expert Craftsmanship</b> – work should appear carefully labored over</li>
</ul>

<hr>

<h2>Deducing the Subtle Reference</h2>

<p>
Before creating the canvas, identify a subtle conceptual reference hidden in the request.
</p>

<p>
The reference becomes the invisible conceptual DNA embedded in:
</p>

<ul>
<li>form</li>
<li>color</li>
<li>composition</li>
<li>pattern</li>
</ul>

<p>
The reference should remain subtle and not explicitly announced.
</p>

<hr>

<h2>Canvas Creation</h2>

<p>
Using the design philosophy and conceptual framework, produce a single-page visual artifact.
</p>

<p>
The artwork should feel like a museum or magazine-quality composition:
</p>

<ul>
<li>Strong visual structure</li>
<li>Limited color palette</li>
<li>Precise spacing and composition</li>
<li>Systematic visual language</li>
<li>Minimal typography</li>
</ul>

<p>
Typography must always serve visual communication.
</p>

<hr>

<h2>Typography Rules</h2>

<ul>
<li>Use thin, design-forward fonts</li>
<li>Fonts should be sourced from the <code>./canvas-fonts</code> directory</li>
<li>Typography may become part of the artwork</li>
<li>Text must remain inside margins</li>
<li>No overlapping elements</li>
</ul>

<p>
Every element must have clear spacing and breathing room.
</p>

<hr>

<h2>Craftsmanship Standard</h2>

<p>
The final design must appear as though it took countless hours to create.
</p>

<ul>
<li>Precise composition</li>
<li>Thoughtful color choices</li>
<li>Careful typography placement</li>
<li>Flawless formatting</li>
</ul>

<p>
The result should be a piece worthy of gallery presentation.
</p>

<hr>

<h2>Output Format</h2>

<p>The skill produces:</p>

<ul>
<li><b>.md</b> – design philosophy</li>
<li><b>.pdf</b> or <b>.png</b> – final artwork</li>
</ul>

<hr>

<h2>Final Refinement</h2>

<p>
Before finishing, refine the design instead of adding new elements.
</p>

<p>
Focus on:
</p>

<ul>
<li>composition harmony</li>
<li>visual balance</li>
<li>clarity and minimalism</li>
</ul>

<p>
The goal is a pristine masterpiece suitable for museum display.
</p>

<hr>

<h2>Multi-Page Option</h2>

<p>
If multiple pages are requested:
</p>

<ul>
<li>Create additional pages within the same visual philosophy</li>
<li>Each page should be distinct but stylistically consistent</li>
<li>Pages may tell a visual narrative</li>
<li>Export as a multi-page PDF or multiple PNGs</li>
</ul>

<p>
Think of the pages as a sequence in a visual art book.
</p>
